IFF-DM Data Miner

Predict the success of a new Experiment or Submission for the different market segments.

Retrieve

Retrieve a dataset that will be used by the different Data Mining modules. These datasets are turned into a generic format having input features and output features. The different datasets that can be retrieved are the outcome of E2E (End-to-End) scenarios.

Learn

IFF-DM prediction is based on several Machine Learning models. These models are tained and tested to predict the success (WIN or LOSS) of a formula for the different market segments. At this moment the prediction is "only" based on the ingredients being used. In the future this prediction can be extended with othe features derived from: ingredient, market, olfactive, project, hedonic.

Predict

Once the IFF-DM Machine Learning models are trained and tested, new data can be classified on WIN or LOSS. This new data is retrieved on-line from BW and will consists of the Experiments or Submissions of the selected project. For each experiment or material belonging to the project a WIN/LOSS classification will be predicted for the different market segments and using different ML models.

Generate

In IFF-DM it is also possible to Generate new Formulas using Monte Carlo techniques. This means that statistical distributions are used on ingredient usage (frequncy and dosage) to come up with new formulas. These new formulas are submitted to the Predict function to derive whether they can be successful.

Basket

Besides Predict it is also possible to Compare the new Experiment or Submissions with other(Winning) Experiments. Shopping Basked techniques are used known from shopping web sites (others also bought). Based on the overlap of used ingredients and dosage, comparable Experiments or Submissions are shown.
